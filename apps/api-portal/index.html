Getting Started

Get up and running with Whistle RPC in minutes. Our endpoints are fully compatible with Solana's JSON-RPC specification.

Base URLs
  • https://rpc.whistle.ninja — Solana RPC endpoint
  • wss://rpc.whistle.ninja — WebSocket endpoint
  • https://dex.whistle.ninja/v1 — DEX & Market Data API
  • wss://dex.whistle.ninja/ws — Real-time price WebSocket
  • https://rpc.whistle.ninja/das — DAS API (NFT/Digital Assets)
// JavaScript/TypeScript - Quick Start npm install @solana/web3.js import { Connection, PublicKey } from '@solana/web3.js'; const connection = new Connection( 'https://rpc.whistle.ninja?api_key=YOUR_API_KEY' ); // Get current slot const slot = await connection.getSlot(); // Get wallet balance const pubkey = new PublicKey('YOUR_WALLET'); const balance = await connection.getBalance(pubkey); console.log(`Balance: ${balance / 1e9} SOL`);
# Python - Quick Start pip install solana from solana.rpc.api import Client from solana.publickey import PublicKey client = Client("https://rpc.whistle.ninja?api_key=YOUR_API_KEY") # Get balance balance = client.get_balance(PublicKey("YOUR_WALLET")) print(f"Balance: {balance['result']['value'] / 1e9} SOL")
# cURL - Direct JSON-RPC curl -X POST https://rpc.whistle.ninja \ -H "Content-Type: application/json" \ -H "X-API-Key: YOUR_API_KEY" \ -d '{ "jsonrpc": "2.0", "id": 1, "method": "getSlot" }'

RPC Methods

Full Solana JSON-RPC 2.0 specification. All standard methods are supported.

Account & Balance Methods
POST getBalance

Returns the SOL balance of a wallet in lamports (1 SOL = 1,000,000,000 lamports).

{ "jsonrpc": "2.0", "id": 1, "method": "getBalance", "params": ["vines1vzrYbzLMRdu58ou5XTby4qAqVRLmqo36NKPTg"] } // Response: { "jsonrpc": "2.0", "result": { "value": 5000000000 }, "id": 1 }
POST getAccountInfo

Returns all account data including owner, lamports, and parsed data.

{ "jsonrpc": "2.0", "id": 1, "method": "getAccountInfo", "params": [ "ACCOUNT_ADDRESS", { "encoding": "jsonParsed" } ] }
POST getMultipleAccounts

Batch fetch multiple accounts in a single request (max 100).

{ "jsonrpc": "2.0", "id": 1, "method": "getMultipleAccounts", "params": [ ["ADDRESS_1", "ADDRESS_2", "ADDRESS_3"], { "encoding": "jsonParsed" } ] }
Token Methods
POST getTokenAccountsByOwner

Get all SPL token accounts owned by a wallet with balances.

{ "jsonrpc": "2.0", "id": 1, "method": "getTokenAccountsByOwner", "params": [ "WALLET_ADDRESS", { "programId": "TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA" }, { "encoding": "jsonParsed" } ] }
POST getTokenLargestAccounts

Get top 20 largest holders of a token.

{ "jsonrpc": "2.0", "id": 1, "method": "getTokenLargestAccounts", "params": ["6Hb2xgEhyN9iVVH3cgSxYjfN774ExzgiCftwiWdjpump"] } // Response includes holder addresses and amounts
POST getTokenSupply

Get total and circulating supply of a token.

{ "jsonrpc": "2.0", "id": 1, "method": "getTokenSupply", "params": ["TOKEN_MINT_ADDRESS"] }
Transaction Methods
POST getSignaturesForAddress

Get recent transaction signatures with status and timestamps.

{ "jsonrpc": "2.0", "id": 1, "method": "getSignaturesForAddress", "params": [ "WALLET_ADDRESS", { "limit": 50, "before": "SIGNATURE" } ] }
POST getTransaction

Get full transaction details including instructions and logs.

{ "jsonrpc": "2.0", "id": 1, "method": "getTransaction", "params": [ "TX_SIGNATURE", { "encoding": "jsonParsed", "maxSupportedTransactionVersion": 0 } ] }
POST sendTransaction

Submit a signed transaction to the network.

{ "jsonrpc": "2.0", "id": 1, "method": "sendTransaction", "params": [ "BASE64_ENCODED_TRANSACTION", { "encoding": "base64", "skipPreflight": false } ] }
POST simulateTransaction

Simulate a transaction without submitting to see potential errors.

{ "jsonrpc": "2.0", "id": 1, "method": "simulateTransaction", "params": [ "BASE64_ENCODED_TRANSACTION", { "encoding": "base64" } ] }

DAS API (Digital Asset Standard)

Query NFTs, tokens, and digital assets with the Metaplex DAS standard. Powered by our own PostgreSQL index - no validator secondary indexes needed.

getAsset

Get metadata for a single NFT/token by mint address.

Get all NFTs owned by a wallet address.

Get all NFTs created by an address. Powered by PostgreSQL index for fast queries.

WebSocket

Real-time subscriptions for account changes, price updates, and live trading data.

Solana RPC WebSocket
// Connect to Solana WebSocket const ws = new WebSocket('wss://rpc.whistle.ninja?api_key=YOUR_KEY'); // Subscribe to account changes ws.send(JSON.stringify({ "jsonrpc": "2.0", "id": 1, "method": "accountSubscribe", "params": ["ACCOUNT_ADDRESS", { "encoding": "jsonParsed" }] })); // Subscribe to logs mentioning a program ws.send(JSON.stringify({ "jsonrpc": "2.0", "id": 2, "method": "logsSubscribe", "params": [{ "mentions": ["PROGRAM_ID"] }, { "commitment": "confirmed" }] })); // Subscribe to slot updates ws.send(JSON.stringify({ "jsonrpc": "2.0", "id": 3, "method": "slotSubscribe" })); // Subscribe to signature confirmation ws.send(JSON.stringify({ "jsonrpc": "2.0", "id": 4, "method": "signatureSubscribe", "params": ["TX_SIGNATURE"] })); ws.onmessage = (event) => { const data = JSON.parse(event.data); console.log('Update:', data); };
DEX Price WebSocket
// Connect to DEX WebSocket (no auth required) const ws = new WebSocket('wss://dex.whistle.ninja/ws'); // Subscribe to token price updates ws.send(JSON.stringify({ "type": "subscribe", "channel": "price", "mint": "6Hb2xgEhyN9iVVH3cgSxYjfN774ExzgiCftwiWdjpump" })); // Subscribe to live trades ws.send(JSON.stringify({ "type": "subscribe", "channel": "trades", "mint": "6Hb2xgEhyN9iVVH3cgSxYjfN774ExzgiCftwiWdjpump" })); // Subscribe to new token listings ws.send(JSON.stringify({ "type": "subscribe", "channel": "newTokens" })); ws.onmessage = (event) => { const { channel, data } = JSON.parse(event.data); if (channel === 'price') { console.log(`Price: $${data.price} (${data.change24h}%)`); } if (channel === 'trades') { console.log(`${data.side}: ${data.amount} @ $${data.price}`); } };

// Recommended error handling async function fetchWithRetry(url, retries = 3) { for (let i = 0; i < retries; i++) { try { const res = await fetch(url); if (res.status === 429) { const wait = res.headers.get('X-RateLimit-Reset') * 1000 - Date.now(); await new Promise(r => setTimeout(r, Math.max(wait, 1000))); continue; } if (!res.ok) throw new Error(`HTTP ${res.status}`); return await res.json(); } catch (err) { if (i === retries - 1) throw err; await new Promise(r => setTimeout(r, 1000 * (i + 1))); } } } // Invalid API key (401) { "error": { "code": 401, "message": "Invalid or expired API key." } } // Token not found (404) { "success": false, "error": "Token not found" }
